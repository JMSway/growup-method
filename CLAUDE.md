# CLAUDE.md — Лонгрид «Grow Up»

## Проект

Mobile-first лендинг (лонгрид) для продажи 1-часовой консультации владельцам языковых школ Казахстана.
Автор: Дамир. Система: «Grow Up». Цель: выход на 5 000 000 ₸/мес.
Продукт: консультация за 4 990 ₸. CTA ведёт в WhatsApp.

## Стек

- Чистый HTML/CSS/JS (без фреймворков, без сборщиков)
- Шрифты: Bebas Neue (заголовки) + Gilroy (body) — локальные, assets/fonts/
- Иконки: Lucide через Iconify Web Component (CDN)
- SVG-иллюстрации: unDraw (цвет #E8651A)
- Тёмная тема по умолчанию, светлая по toggle

## Секции лендинга (12 блоков)

1. **HERO** — full-screen (min-height: 100dvh), safe-area, CTA в thumb zone
2. **ДО/ПОСЛЕ** — кейс Айнуры (Караганда), скриншоты переписок + чеки
3. **ЭКСПЕРТ + ПРОБЛЕМЫ** — Дамир, 2 года опыта, 3 боли рынка
4. **МЕХАНИЗМ** — Шаг 1 (Алмаз среди камней), Шаг 2 (ТПУ), Шаг 3 (Продажи на пробном)
5. **РЕЗУЛЬТАТ** — итог: 5 000 000 ₸ за 30 дней + визуал до/после
6. **«КАЖЕТСЯ ПРОСТЫМ»** — 3 инсайта (2 года опыта)
7. **ЧТО БУДЕТ НА КОНСУЛЬТАЦИИ** — 6 пунктов
8. **ЦЕНА** — якоря 499 990 → 49 990 → **4 990 ₸** (для 10 школ)
9. **БОНУСЫ** — 3 PDF/таблицы, итого 75 970 ₸ бесплатно
10. **ГАРАНТИЯ** — безусловный возврат, «Дамир, хочу возврат»
11. **FAQ** — 3 вопроса (accordion), после каждого CTA
12. **ФИНАЛЬНЫЙ CTA** — цена + счётчик мест + кнопка WhatsApp

## Правила разработки

### Общие
- Mobile-first: 375–430px, max-width: 480px для контейнера
- Только вертикальный скролл
- Отступы: 16–20px по бокам
- Карточки: border-radius 16px, padding 32px 20px
- Hero ОБЯЗАТЕЛЬНО full-screen (100dvh + safe-area)

### Иконки
- Только Lucide (outline), формат `lucide:icon-name`
- Без emoji — только иконки
- Размеры: 20–24px, currentColor по умолчанию
- CTA иконка слева от текста
- Полный маппинг — см. ICONS.md

### CTA-кнопка (глобальный компонент)
- Фон: #E8651A, текст: #FFFFFF, border-radius: 12px
- Иконка WhatsApp белая слева
- Текст: «ХОЧУ НА КОНСУЛЬТАЦИЮ» (caps, Gilroy Bold)
- Hover: #F07535, Active: #C9530F

### Тема
- `data-theme="dark"` по умолчанию на `<html>`
- Toggle: fixed, top: 16px, right: 16px, 36x36px, border-radius: 50%
- Переключение через JS, transition 0.3s ease

### Запрещено
- Чистый чёрный #000000
- Чистый белый #FFFFFF
- Зелёные/синие/фиолетовые кнопки
- Emoji вместо иконок
- Горизонтальный скролл
- Длинные абзацы
- Изменение структуры воронки
